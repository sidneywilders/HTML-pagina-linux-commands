Handige Linux Commando's

CommandoBeschrijving
pwdPrint working directory
lsList bestanden
ls -lLaat zien wanneer bestanden zijn aangemaakt
ls -aToont alle bestanden (ook verborgen)
cdVerander directory
cd /var/homeGa naar opgegeven map
cd ..Ga één map omhoog
touchMaak een lege file
mkdirMaak een nieuwe map
rmVerwijder bestand of map
cpKopieer bestand of map
mvVerplaats bestand of map
manToont uitleg over een commando
treeToont folderstructuur
nanoCommand line teksteditor
catLaat inhoud van bestand zien
lessBekijk inhoud per pagina
headLaat bovenkant van een bestand zien
tailLaat onderkant van een bestand zien
clearMaak terminal leeg
ctrl + xSluit nano af
qSluit less viewer af
head -n 5Toon de eerste 5 regels
tail -n 5Toon de laatste 5 regels
tail -fVolg een bestand live
sudo less /var/log/syslogOpen systeemlogboek
sudo tail /var/log/syslogBekijk laatste regels van logboek
sudo head /var/log/syslogBekijk eerste regels van logboek
sudo cat /var/log/syslogToon volledige inhoud van logbestand
